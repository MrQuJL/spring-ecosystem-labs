# 创建 Spring Boot 最小化演示项目

我将在 `/Users/qujianlei/Documents/GitHub/spring-ecosystem-labs/spring-boot-basics` 目录下创建一个名为 `spring-boot-minimal-demo` 的项目。

## 技术栈

* **Spring Boot**: 2.7.14 (参考现有项目的版本)

* **Java**: 1.8

* **构建工具**: Maven

## 实施步骤

### 1. 创建项目目录结构

创建基本的 Maven 目录结构：

* `spring-boot-minimal-demo/`

  * `src/main/java/com/github/jeremy/minimal/`

  * `src/main/resources/`

### 2. 创建 pom.xml

配置最简化的 Maven 依赖，仅包含 `spring-boot-starter-web` 以便提供基础的 HTTP 访问验证。

### 3. 编写启动类

创建 `MinimalApplication.java`，包含 `@SpringBootApplication` 注解和 `main` 方法。
同时添加一个简单的 `@RestController` 映射，以便启动后可以访问 `http://localhost:8080/hello` 来验证项目是否运行正常。

### 4. 编写配置文件

创建 `src/main/resources/application.yml`，设置基础的项目名称。

## 验证

项目创建完成后，可以进入该目录运行 `mvn spring-boot:run` 来启动项目。
访问 `http://localhost:8080/hello` 确认返回 "Hello, Spring Boot Minimal!"。
